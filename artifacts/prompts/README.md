# Prompt Artifacts

This directory stores prompt templates used by the reported experiments. The
separately compiled supplementary material reproduces the full prompt text.
The artifact preserves the executed wording verbatim for reproducibility.

- `llm_fault_location_prompt.md`: JSON-output prompt used by the LLM fault
  localization baselines. Its legacy wording is frozen because the archived
  outputs were generated with this exact text.
- `glm5_repair_prompt.md`: system and user templates used by the controlled
  GLM-5 repair comparison.
