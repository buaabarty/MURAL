# Prompt Artifacts

This directory stores the complete prompt templates used by the reported
experiments. The supplementary material reproduces the system prompt and
summarizes the user template; the artifact preserves the executed wording
verbatim.

- `llm_fault_location_prompt.md`: JSON-output prompt used by the LLM fault
  localization baselines. Its legacy wording is frozen because the archived
  outputs were generated with this exact text.
- `glm52_repair_prompt.md`: system and user templates used by the controlled
  GLM-5 repair comparison. The filename retains the endpoint's legacy request
  alias for traceability.
